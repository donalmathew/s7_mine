Common patterns:
```
// Categories
Dog(Fido)                     // Fido is a dog

// Properties
Tall(John)                    // John is tall

// Relations
ParentOf(Mary, John)          // Mary is parent of John

// Rules
∀x(Dog(x) → Animal(x))        // All dogs are animals

// Complex statements
∀x(Student(x) ∧ StudiesHard(x) → WillPass(x))
// All students who study hard will pass
```
